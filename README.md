# 猫咪领养
## 前言
这是一个猫咪领养小程序，主要通过用户在线填写表单，发布流浪猫信息；其他用户通过搜索查找，到心仪的猫咪，完善个人信息后，进行猫咪领养，领养前需要先查看是否符合领养要求，比如是否同城，已封窗等等；确认无误后，进行申请，确认无误后可以获得相关联系方式；

## 技术栈
  - 微信小程序
  - less + ts
  - 微信云开发
  - vant/weapp组件库
